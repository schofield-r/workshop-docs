HTML

HTML stands for Hyper Text Markup Language. It os the standard markup language for creating web pages and is used to describe the structure of the page. The elements inform the browser how to display the web page and with a bit of styling with CSS it makes a pretty neat site

A simple HTML document: 

    
    
      <!DOCTYPE html>
      <html>
      <head>
      <title>Page Title</title>
      </head>
      <body>
      
      <h1>My First Heading</h1>
      <p>My first paragraph.</p>
      
      </body>
      </html>

What is an HTML Element? An HTML element is defined by a start tag, some content, and an end tag: 


    
    
    <tagname> Content goes here... </tagname>
